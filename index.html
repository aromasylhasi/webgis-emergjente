Kopjo URL-në dhe shto si Shtresë Vektori (HTTP) në QGIS. Serveri duhet të jetë aktiv (python -m http.server 8080).
Shërbimi WMS kërkon server të dedikuar si GeoServer ose QGIS Server. Projekti aktual nuk ka WMS server — përdorni WFS/GeoJSON si alternativë.
